> Plataforma interativa de mapas colaborativos com POIs, progresso de usuário e inteligência artificial

- 🏗️ **Arquitetura de Microserviços** (NestJS + FastAPI + Next.js)
- 📍 **Sistema Geoespacial** com PostGIS e MapLibre GL
- 🤖 **Features de IA**: Deduplicação, auto-tagging, otimização de rotas
- ⚡ **Alta Performance**: Cache Redis, clustering server-side, p95 < 95ms
- 🔒 **Segurança**: JWT, RBAC, rate limiting
- 🐳 **Docker Compose** com setup automatizado

**Stack**: TypeScript, Next.js, NestJS, FastAPI, PostgreSQL, PostGIS, Redis, Docker
